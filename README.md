# Duendes del Uruguay - Backend Services

Servicios de backend para las tiendas Shopify de Duendes del Uruguay.

## Que hace este proyecto

1. **Sincronizacion de inventario** entre la tienda US y UY
2. **Emails automaticos** (18 emails en 4 secuencias post-compra)

## Stack
- Next.js 14.2.0 (Vercel)
- Shopify Admin API
- Resend (emails)
- Vercel KV (cola de emails)
